# Cybersecurity-Toolkit-Python
Cybersecurity Toolkit is a Python-based, menu-driven security tool that helps users identify common threats. It features a URL Scanner for phishing indicators and a Password Strength Checker that evaluates password security using length, complexity, and pattern analysis. Runs fully offline with no external libraries.
Cybersecurity Toolkit (Python)
Overview
This project is a menu-driven cybersecurity toolkit built with Python. It helps users identify common security risks using lightweight, offline analysis techniques.
Features
URL Security Scanner: Analyzes URLs for phishing indicators such as unusual length, excessive hyphens, IP-based URLs, and suspicious keywords.
Password Strength Checker: Evaluates password strength based on length, character diversity, and predictable patterns. Passwords are not stored.
How It Works
Users select a tool from a menu. Each tool runs independently and returns to the menu.
Requirements
Python 3.x
No external libraries
Offline usage
How to Run
Run the main Python file and follow on-screen instructions.
Use Cases
Cybersecurity learning
Checking suspicious links
Password strength evaluation
